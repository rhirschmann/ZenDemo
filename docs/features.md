# Features

Zensical provides a modern, fast, and flexible documentation experience.  
This page highlights some of the key features enabled in your configuration.

---

## 🚀 Navigation

- Automatic navigation from folder structure
- Optional explicit navigation via `zensical.toml`
- Breadcrumbs (`navigation.path`)
- Instant navigation (`navigation.instant`)
- Instant prefetching (`navigation.instant.prefetch`)
- Footer navigation (`navigation.footer`)
- Section indexes (`navigation.indexes`)

---

## ✍️ Authoring Enhancements

Zensical includes many Markdown extensions:

- Admonitions  
- Footnotes with tooltips  
- Code annotations  
- Code copy button  
- Code line selection  
- Task lists  
- Tabbed content  
- Tooltips  
- Mermaid diagrams  
- Emoji support  
- Math rendering (Arithmatex)

---

## 🎨 Themes & Styling

Zensical ships with two theme variants:

- **modern** (default)
- **classic**

You can switch themes in your `zensical.toml`:

```toml
[project.theme]
variant = "classic"
```
