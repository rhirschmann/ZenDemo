# Features

Zensical provides a modern, fast, and flexible documentation experience.  
This page highlights some of the key features enabled in your configuration.

---

## 🚀 Navigation

- Automatic navigation from folder structure
- Optional explicit navigation via `zensical.toml`
- Breadcrumbs (`navigation.path`)
- Instant navigation (`navigation.instant`)
- Instant prefetching (`navigation.instant.prefetch`)
- Footer navigation (`navigation.footer`)
- Section indexes (`navigation.indexes`)

---

## ✍️ Authoring Enhancements

Zensical includes many Markdown extensions:

- **Admonitions** — Callout boxes for notes, warnings, tips
- **Footnotes** — References with tooltips  
- **Code annotations** — Inline code documentation
- **Code copy button** — One-click code block copying
- **Code line selection** — Highlight specific lines
- **Task lists** — Interactive checkboxes
- **Tabbed content** — Organize content in tabs
- **Tooltips** — Hover definitions
- **Mermaid diagrams** — Flowcharts and sequences
- **Emoji support** — Unicode emoji rendering
- **Math rendering** — Arithmatex LaTeX support

---

## 🎨 Themes & Styling

Zensical ships with two theme variants:

- **modern** (default) — Contemporary design
- **classic** — Clean, traditional look

Switch themes in your `zensical.toml`:

```toml
[project.theme]
variant = "classic"
```
