# Welcome to the Zensical Demo

This demo site shows how to build clean, modern documentation using **Zensical**.

Zensical gives you:

- Instant navigation
- Modern UI
- Markdown-based authoring
- Powerful extensions
- Fast builds

> Tip: You can edit this page and see changes instantly using `zensical serve`.

## 🚀 Get Started in 30 Seconds

```bash
# Clone and setup
git clone https://github.com/rhirschmann/ZenDemo.git
cd ZenDemo
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install zensical

# Run development server
zensical serve
```

Visit `http://127.0.0.1:8000` and start editing docs!
