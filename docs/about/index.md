# About Zensical

Zensical is a modern, fast static site generator designed specifically for beautiful documentation.

## What is Zensical?

Zensical combines the simplicity of Markdown with powerful modern web technologies to create stunning documentation sites. It's built for:

- **Documentation projects** — User guides, API docs, tutorials
- **Knowledge bases** — Internal docs, wikis, learning materials
- **Project sites** — Showcases, portfolios, landing pages
- **Technical writing** — Blog posts, guides, specifications

- **Instant Navigation** — Pre-loads pages for instant navigation
- **Dark Mode** — Automatic or manual theme switching
- **Search** — Full-text search with instant results
- **Responsive Design** — Works perfectly on any device
- **Markdown Extensions** — Admonitions, tabs, code annotations, and more
- **Mermaid Diagrams** — Create flowcharts and sequences
- **Math Support** — LaTeX rendering for formulas
- **Customizable** — Themes and extensive configuration

## Getting Started

### Installation

```bash
pip install zensical
```

### Create a New Site

```bash
zensical init my-docs
cd my-docs
```

### Run Development Server

```bash
zensical serve
```

Visit `http://127.0.0.1:8000` in your browser.

### Build for Production

```bash
zensical build
```
